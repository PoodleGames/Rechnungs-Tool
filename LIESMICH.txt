RECHNUNGSTOOL — SO GEHT'S
==========================

SO STARTEST DU DAS TOOL
------------------------
1. Doppelklick auf die Datei "Rechnungstool.html"
   (öffnet sich automatisch in deinem Browser, z.B. Microsoft Edge)

2. Beim ersten Start erscheint ein Fenster:
   "Rechnungstool mit deinem Ordner verbinden"
   → Klicke auf "Ordner auswählen"
   → Wähle den Ordner aus, in dem diese Datei liegt
     (also den Ordner "Rechnungstool" bzw. wie auch immer du ihn
     genannt hast — NICHT einen Unterordner, sondern den, der
     "Rechnungstool.html", "data" und "output" enthält)
   → Fertig! Das musst du nur EIN EINZIGES MAL machen.

3. Ab jetzt kannst du ganz normal Rechnungen erstellen.

DAS WAR'S SCHON. Keine Installation, kein Programm herunterladen,
nichts weiter nötig.


WAS DIE DREI SEITEN MACHEN
----------------------------
- Rechnungstool.html  → Rechnung erstellen und bearbeiten
- Kunden.html         → Kunden anlegen, bearbeiten, löschen
- Artikel.html        → Leistungen/Artikel anlegen, die du in
                         Rechnungen einfügen kannst

Zwischen den Seiten kannst du jederzeit über die Knöpfe oben
rechts wechseln.


WIE DU EINE RECHNUNG SPEICHERST
----------------------------------
Unten rechts gibt es drei Knöpfe:

- "HTML speichern"
  → lädt die Rechnung als Datei herunter (in deinen normalen
    Download-Ordner), so wie du es vielleicht schon kennst.

- "Im Kundenordner ablegen"
  → DAS ist der wichtige Knopf für echte, fortlaufend
    nummerierte Rechnungen! Vergibt automatisch die nächste
    Rechnungsnummer (R-0001, R-0002, R-0003, ...) und speichert
    die Rechnung automatisch im richtigen Kundenordner unter
    "output". Für jeden Kunden wird automatisch ein eigener
    Ordner angelegt.

- "Als PDF exportieren"
  → öffnet den normalen Druckdialog deines Browsers. Dort kannst
    du "Als PDF speichern" auswählen.


WO DEINE DATEN LIEGEN
------------------------
Alles bleibt auf deinem eigenen Rechner, in diesem Ordner:

  data/
    kunden.json         → all deine Kunden
    artikel.json         → all deine Leistungen/Artikel
    einstellungen.json   → deine Firmendaten, Rechnungsnummern-Zähler

  output/
    K-00001_Kundenname/
      Rechnung_R-0001_2026-06-30.html
      Rechnung_R-0005_2026-07-12.html
    K-00002_Anderer_Kunde/
      Rechnung_R-0002_2026-06-30.html

TIPP: Mache ab und zu eine Sicherheitskopie von diesem ganzen
Ordner (z.B. auf einen USB-Stick kopieren), damit nichts
verloren geht.


HÄUFIGE FRAGEN
-----------------
"Es kommt eine Sicherheitswarnung vom Browser, ist das normal?"
→ Ja. Der Browser fragt einmalig nach, ob diese Webseite auf
   deine Dateien zugreifen darf. Das ist normal und gewollt —
   nur so kann das Tool deine Rechnungen direkt auf deinem
   Rechner speichern. Klicke einfach auf "Zulassen" bzw.
   "Ordner auswählen".

"Ich nutze Firefox, es funktioniert nicht."
→ Bitte Microsoft Edge oder Google Chrome verwenden. Edge ist
   auf jedem Windows-Rechner bereits vorinstalliert (blaues
   "e"-Symbol). Firefox unterstützt die nötige Technik aktuell
   nicht.

"Muss ich jedes Mal den Ordner neu auswählen?"
→ Nein, nur beim allerersten Start. Danach merkt sich der
   Browser deine Auswahl automatisch.

"Kann ich den Ordner verschieben oder umbenennen?"
→ Ja, das geht. Du musst danach nur einmal erneut den Ordner
   auswählen (das Verbinden-Fenster erscheint dann automatisch
   wieder einmalig).
